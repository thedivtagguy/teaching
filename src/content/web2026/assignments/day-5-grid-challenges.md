# Grid Challenges

Take a look at the grid designs below. Each is a layout you'll plausibly want for your own website. Grid lines are drawn to help you see the structure.

**Solve any two.** Two layouts done thoughtfully will teach you more than five done frantically, which is why I'm only asking for two. Write the markup first, then style incrementally in a separate CSS file. Start every solution with `grid-template-areas`: name your rooms, then place your elements. Reach for `fr` and column counts only when names aren't enough. I'm more interested in you figuring out *how to think about* the layout than in a pixel-perfect copy, so put your own content in, use your own colors and fonts.

**Due at the start of Day 7 (Monday). Time-box: 90 minutes.** You have Friday evening and Sunday; **Saturday night is off**, because by then the terminal will have earned you the rest. Stuck past the box? Stuck-note it; Monday-me will help.

Once you are done, push to Netlify, either both on one page **or learn how to create two separate pages and link them** (recommended: your personal site will need exactly this skill in a week), and submit the link.
