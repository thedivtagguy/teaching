# Fit to Print

This exercise has two tracks, of which you need to choose one.

- **Fairy tale**: one of Hans Christian Andersen's stories.
- **Newspaper**: a page or article from a historical newspaper.

The full brief, both source collections, and the [image pools](/web2026/day-4-typesetting-a-page#image-pools) are on the [Day 4 page](/web2026/day-4-typesetting-a-page). I've spelled the expectations out in full on the same page. You need to demonstrate using flexbox alongside the other tools, intentional use of fonts, images, one technique borrowed from [Jen Simmons' labs](https://labs.jensimmons.com/), and one idea stolen, oops sorry, *borrowed*, from [CSS Zen Garden](https://csszengarden.com). 

| Order of work                                                                            |
| ---------------------------------------------------------------------------------------- |
| Pick your track + source, design reviewed by me                                          |
| Roughly typeset the page in vanilla HTML & CSS and get the structure into the index.html |
| Polish the CSS, your layouts and styling                                                 |
| Deploy to Netlify                                                                        |
|                                                                                          |

The [Day 5 page](/web2026/day-5-the-make-day) has the starter kit that you can clone to start your page with, and also shows you some common traps and their fixes. There is also a section on asking LLMs for help in this exercise. You will be expected to discover new CSS properties and how to use them, and in such cases it is acceptable (and recommended) learning how to ask an LLM a question that actually teaches you something. Lean on it while you build, but don't make it build things for you. 

Simple and well-executed beats complex and broken, every time. [Even whitespace can look good](https://labs.jensimmons.com/2017/01-021.html). This is a chance to demonstrate your understanding of color, hierarchy and type from your previous courses.
